# VodPulse

VodPulse is a browser extension for Twitch that visualizes chat activity density directly on the VOD timeline.

It allows you to instantly identify the most engaging, funny, or loud moments of a stream without watching the entire multi-hour recording.

## Installation

Since this project is currently in development and not available on the Chrome Web Store, it must be installed manually:

1.  Clone or download this repository.
2.  Open a Chromium-based browser (Chrome, Edge, Brave, Opera).
3.  Navigate to `chrome://extensions/`.
4.  Enable **Developer mode** in the top right corner.
5.  Click **Load unpacked**.
6.  Select the folder containing the project files (`manifest.json`, `content.js`, `styles.css`).

## Usage

1.  Open any past broadcast (VOD) on Twitch.tv.
2.  Locate the **"VodPulse"** button in the player control bar (left side, next to the volume/pause button).
3.  Click the button.
4.  Wait for the chat download to complete (progress is shown on the button).
5.  The activity graph will appear over the timeline.
